# 🌍 Panduan Deploy ke Netlify (Netlify Drop)

Karena komputer Anda saat ini tidak memiliki Node.js/NPM atau Git terinstal secara global, cara termudah dan tercepat untuk mempublikasikan game Anda (Ludo King & Football Manager) ke internet adalah menggunakan layanan **Netlify Drop** (drag-and-drop gratis).

Ikuti 3 langkah mudah berikut untuk mengaktifkan game Anda secara online:

---

### 🚀 Langkah 1: Buka Netlify Drop
Buka browser Anda dan kunjungi tautan berikut:
👉 [**https://app.netlify.com/drop**](https://app.netlify.com/drop)

---

### 📂 Langkah 2: Unggah Folder Game
1. Buka File Explorer di komputer Anda.
2. Cari folder game Anda yang berada di:
   `C:\Users\esra7\Documents\program game`
3. Tarik (*drag*) dan lepaskan (*drop*) seluruh folder **`program game`** ke area kotak biru putus-putus bertuliskan **"Drag and drop your site folder here"** di situs Netlify tersebut.

---

### 🔗 Langkah 3: Selesai!
1. Netlify akan mulai mengunggah file Anda (proses ini memakan waktu kurang dari 10 detik).
2. Setelah selesai, Anda akan diberikan tautan web resmi yang aktif secara otomatis (contoh: `https://bright-crepe-12345.netlify.app`).
3. Anda sekarang dapat mengakses kedua game Anda secara langsung melalui tautan tersebut:
   * **Ludo Cyber King**: `https://<nama-situs-anda>.netlify.app/index.html` (atau cukup buka domain utamanya)
   * **Football Manager**: `https://<nama-situs-anda>.netlify.app/manager.html` (bisa juga diakses dengan menekan tombol **"KARIR FOOTBALL MANAGER ⚽"** di halaman awal Ludo).

---

> [!TIP]
> Jika Anda mendaftar akun Netlify (gratis), Anda dapat mengubah nama subdomain acak tersebut (misal menjadi `https://game-football-manager.netlify.app`) secara permanen melalui menu **Site Configuration** -> **Change Site Name**.
